# Weather Service

Простой HTTP-сервис для получения и хранения данных о погоде.

---

### Описание

Сервис предоставляет REST API для получения текущей температуры по названию города. Данные автоматически обновляются каждые 10 секунд через планировщик задач.

---

### Технологии

- Go 1.24.5
- Chi router
- Open-Meteo API для получения погодных данных
- Geocoding API для определения координат городов

---

### Запуск

```bash
go run cmd/server/main.go
```

Сервис запускается на порту `:3000`.

---

### Пример запроса

#### GET /{city}

Возвращает массив с температурными данными для указанного города.

**Запрос:**
```
GET /moscow
```

**Ответ:**
```json
[
  {
    "timestamp": "2025-12-14T12:30:00Z",
    "temperature": -5.2
  }
]
```
