# Weather service - получение данных о погоде в любом городе.

## Технологии

- Go 1.24.5
- Chi router
- Geocoding API для определения координат городов
- Open-Meteo API для получения погодных данных

## Запуск

```bash
go run cmd/server/main.go
```

---

Сервис запускается на порту `:3000`.

## Пример запроса GET /{city}

Возвращает массив с температурными данными для указанного города.

**Запрос**
```
GET /moscow
```

**Ответ:**
```json
[
  {
    "timestamp": "2025-12-14T12:30:00Z",
    "temperature": -5.2
  }
]
```
