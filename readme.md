## Weather Service - получение информации о температуре в любом городе

Простой HTTP-сервис для получения и хранения данных о погоде. Работает с двумя сторонними API. 

## Технологии

- Go 1.24.5
- Open-Meteo API для получения погодных данных
- Geocoding API для определения координат городов

### Запуск

```bash
go run cmd/server/main.go
```

Сервис запускается на порту `:3000`.

## Пример запроса GET /{city}

Возвращает массив с температурными данными для указанного города.

**Запрос:**
```
GET /moscow
```

**Ответ:**
```json
[
  {
    "timestamp": "2025-12-14T12:30:00Z",
    "temperature": -5.2
  }
]
```  

## Структура проекта

```
cmd/server/     - запуск сервера
handlers/       - обработка запросов и ответов
sideAPI/
  geocoding/    - получение координат
  openmeteo/    - получение температуры
```
